# 3d-embeddable
CDN-Repo to hold our packaged embeddable.

# For nextjs
```js
<div
        className="aviana-calculator-widget"
        data-url="https://64565f685f9a4f236142355a.mockapi.io/forms/"
        data-form-id="form_id_123"
        data-modal-title="Add modal title here"
        data-modal-description="Add the modal description here"
        ></div>
        
        {/* <!--This is the embedable to the widget--> */}
        <Script src="https://cdn.jsdelivr.net/gh/GeckoDevelopment/3d-embeddable@main/dist-widget/index.js"></Script>
        <link
        rel="stylesheet"
        type="text/css"
        href="https://cdn.jsdelivr.net/gh/GeckoDevelopment/3d-embeddable@main/dist-widget/index.css"
/>
```
